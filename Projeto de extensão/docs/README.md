# Documentação do Site do Projeto de extensão de Diagnósticos Socioeconômicos do Vale do Mamanguape ODS.

Site simples para divulgar o projeto de extensão **Diagnóstico Socioeconômico dos Municípios da Região Metropolitana do Vale do Mamanguape para a Territorialização dos Objetivos de Desenvolvimento Sustentável**. 

Apresentado no: **XIII Encontro Unificado de Ensino, Pesquisa e Extensão da UFPB**
- **Data:** 02 e 04 de dezembro  
- **Local:** UFPB – Campus IV (Rio Tinto e Mamanguape)

O portal reúne informações institucionais, objetivos, metodologia, resultados esperados, relatórios e materiais derivados das atividades realizadas no âmbito do projeto, oferecendo uma visualização integrada dos dados e ações desenvolvidas.

> **Link para a versão online (GitHub Pages):** https://joaoboscoduarte.github.io/ENEXExtensaoODS/

## Detalhes do projeto

- **Natureza:** Projeto de Extensão Universitária  
- **Período de Execução:** 2024 – 2025  
- **Departamento Responsável:** CCAE — Departamento de Ciências Sociais Aplicadas  
- **Instituição:** Universidade Federal da Paraíba (UFPB)  
- **Campus:** Campus IV – Rio Tinto e Mamanguape
